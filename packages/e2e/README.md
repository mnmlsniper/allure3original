## Overview

The package includes E2E tests for Allure packages.

## Run

Install the packages' dependencies:

```shell
yarn install
```

Don't forget to build all the packages which will be tested (run the command in the root directory):

```shell
yarn build
```

Run the tests:

```shell
yarn test
```
